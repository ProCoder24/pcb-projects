\# ESP32 USB-C Development Board



\## Overview

Custom ESP32-S3 Mini-based PCB designed for USB-C power and communication.



\## Features

\- ESP32-S3-MINI module

\- USB-C interface (power + data)

\- On-board voltage regulation

\- Programming interface (UART / USB)
